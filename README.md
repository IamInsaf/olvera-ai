# Olvera AI – Career Guidance Platform

## Overview
Olvera AI is an AI-powered career guidance platform designed to help students make informed decisions about their future. The system analyzes user inputs such as interests, skills, and goals to generate personalized career recommendations.

## Problem
Many students lack clarity when choosing careers due to limited guidance and lack of personalized insights.

## Solution
Olvera uses AI-based evaluation to provide:
- Personalized career suggestions
- Skill-based recommendations
- Structured self-assessment

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Firebase (Authentication, Firestore)
- AI Integration: LLM-based recommendation logic

## Features
- User authentication system
- Structured career assessment flow
- AI-generated suggestions
- Scalable cloud-based backend

## How It Works
1. User completes a structured assessment
2. Data is processed using AI logic
3. System generates career recommendations
4. Results are presented in a simple interface

## Current Status
In development – core system architecture and AI logic under implementation

## Future Improvements
- Advanced AI models for deeper analysis
- Dashboard with analytics and insights
- Gamified assessment experience
- School integration (Olv Alpha)

## Vision
To build a scalable AI platform that provides accurate, accessible, and personalized career guidance to students worldwide.
